解析动态json

jsonC#

动态JSON

在做项目时遇到接口返回的json节点是动态了，如下： 

{
    page:1,
    total:2,
    proList:[{
        name:"text",
        propvalue:[{
            "12321":"asdfdsf"
        }]
    },{
        name:"porasdf",
        propvalue:[{
            "1123213":"asdfdsf"
        }]
    }]
}

以上的json中，如果定义一个实体类是没有办法去解析出来了，其中propvalue里的节点是动态了。

解析json

解析采用的dll是Newtonsoft.Json.dll,其中查看源码发现的有jObject和jArray可以返回解析出来，开始解析： 

//读取保存的txt文件，這裏是以上的json字符串
StreamReader sr6 = File.OpenText(@"D:\text.txt");
string restOfStream = sr6.ReadToEnd();

//开始解析
JObject jobject = (JObject)JsonConvert.DeserializeObject(restOfStream);
foreach (JProperty item in jobject.Properties())
{
    if (item.Name == "proList")
    {
       JArray arr = (JArray)item.Value;
         foreach (var ja in arr) {
             JObject ob =(JObject) ja;
              foreach (JProperty j in ob.Properties()) {
                  //Console.WriteLine(j.Name+"\t"+j.Value);
             }
         }
    }
    else
    {
       //Console.WriteLine(item.Name + "\t" + item.Value);
    }
}

如果是json是数组的就采用JArray进行解析，一般对象就用JObject解析，的每个节点可以根据JProperty得到。
